# AutoCheck 🚗🔧

**AutoCheck** foi desenvolvida em parceria com a **Porto Seguro** para otimizar a gestão de agendamentos e diagnósticos automotivos. A aplicação permite que usuários façam agendamentos de serviços e acompanhem diagnósticos de veículos de maneira eficiente.

## Estrutura do Projeto

- **jdbc/**: Contém o arquivo `.jar` do **ojdbc** para conexão com o banco de dados Oracle e o script SQL para criação das tabelas.
- **.env.sample**: Arquivo de configuração que deve ser duplicado e renomeado para `.env`, onde serão inseridas as credenciais do banco de dados.
